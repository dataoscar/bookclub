Mediator Pattern
=

* Defines an object encapsulates interaction
* Allows loose coupling 
* Acts as a "communication" gateway

Participants and Implementation
-
* Mediator Interface  
* Mediator Implementation  
* Colleague ( objects that communicate through mediator )
